C-pong
---------

An imitation of the classic 'Pong' game, written in C and SDL
	
	- after compilation, place the excecutable in /run
	- run using the terminal, as the scores are printed out to it
	- /run contains linux excecutable- feel free to delete it after compilation
Rules:
	- use the mouse to control the player paddle, located by the left edge of the window
	- first to 7 points wins
